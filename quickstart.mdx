This guide will get you productive with fzf in under 5 minutes. We'll cover basic usage, shell integration, and practical examples.

<Note>
Make sure you've [installed fzf](/installation) before proceeding with this guide.
</Note>

## Basic Usage

<Steps>
  <Step title="Your first fzf command">
    List files in the current directory and select one:
    
    ```bash
    ls | fzf
    ```
    
    - Type to search (fuzzy matching works automatically)
    - Use arrow keys or `CTRL-J`/`CTRL-K` to navigate
    - Press `Enter` to select
    - Press `ESC` or `CTRL-C` to cancel
  </Step>
  
  <Step title="Open a file in your editor">
    Select a file and open it in vim:
    
    ```bash
    vim $(fzf)
    ```
    
    Without any input, fzf automatically traverses the current directory to find files.
    
    <Tip>
    Use `fzf --print0 | xargs -0 -o vim` for a more robust solution that handles spaces in filenames.
    </Tip>
  </Step>
  
  <Step title="Try multi-select">
    Select multiple files using TAB:
    
    ```bash
    # Select multiple files and open in vim
    vim $(fzf -m)
    ```
    
    - Press `TAB` to mark items
    - Press `Shift-TAB` to unmark
    - Press `Enter` to confirm selection
  </Step>
  
  <Step title="Use the preview window">
    Preview file contents while searching:
    
    ```bash
    fzf --preview 'cat {}'
    ```
    
    For better syntax highlighting, use `bat`:
    
    ```bash
    fzf --preview 'bat --color=always --style=numbers {}'
    ```
    
    <Tip>
    Press `CTRL-/` to toggle the preview window on and off.
    </Tip>
  </Step>
</Steps>

## Search Syntax Quick Reference

fzf's fuzzy matching is powerful. Here are the essential search patterns:

<CodeGroup>
```bash Fuzzy Match
# Type characters in any order
fzf
# Type: "fz" matches "fzf", "fuzzy", "frozen", etc.
```

```bash Exact Match
# Use single quotes for exact matching
fzf
# Type: 'fzf matches only "fzf"
```

```bash Prefix Match
# Use ^ for start of line
fzf
# Type: ^fzf matches "fzf-test" but not "test-fzf"
```

```bash Suffix Match
# Use $ for end of line
fzf
# Type: .txt$ matches "readme.txt" but not "txt.md"
```

```bash Inverse Match
# Use ! to exclude
fzf
# Type: !test excludes items containing "test"
```
</CodeGroup>

## Shell Integration

Shell integration provides three powerful key bindings that work in any shell:

<Steps>
  <Step title="CTRL-T: Find files and directories">
    Press `CTRL-T` at the command line to open fzf and select files/directories:
    
    ```bash
    vim    # Press CTRL-T here to select files
    ```
    
    Customize with environment variables:
    
    ```bash
    # Add to ~/.bashrc or ~/.zshrc
    export FZF_CTRL_T_OPTS="
      --preview 'bat -n --color=always {}'
      --bind 'ctrl-/:change-preview-window(down|hidden|)'"
    ```
  </Step>
  
  <Step title="CTRL-R: Search command history">
    Press `CTRL-R` to search your command history:
    
    ```bash
    # Press CTRL-R and start typing to search history
    ```
    
    Useful tips:
    - Press `CTRL-R` again to toggle chronological order
    - Press `ALT-R` to toggle "raw" mode
    - Press `CTRL-/` to toggle line wrapping
    
    Customize the appearance:
    
    ```bash
    # Add to ~/.bashrc or ~/.zshrc
    export FZF_CTRL_R_OPTS="
      --preview 'echo {}' --preview-window up:3:hidden:wrap
      --bind 'ctrl-/:toggle-preview'
      --bind 'ctrl-y:execute-silent(echo -n {2..} | pbcopy)+abort'
      --color header:italic
      --header 'Press CTRL-Y to copy command into clipboard'"
    ```
  </Step>
  
  <Step title="ALT-C: Change directory">
    Press `ALT-C` to cd into any directory:
    
    ```bash
    # Press ALT-C to select and cd into a directory
    ```
    
    Add a tree preview:
    
    ```bash
    # Add to ~/.bashrc or ~/.zshrc
    export FZF_ALT_C_OPTS="
      --preview 'tree -C {} | head -200'"
    ```
  </Step>
</Steps>

## Fuzzy Completion

Fuzzy completion is triggered with `**` followed by TAB:

<Steps>
  <Step title="File and directory completion">
    ```bash
    # Files under current directory
    vim **<TAB>
    
    # Files under parent directory
    vim ../**<TAB>
    
    # Files matching a pattern
    vim ../fzf**<TAB>
    
    # Directories only
    cd **<TAB>
    ```
  </Step>
  
  <Step title="Process completion">
    ```bash
    # Kill processes with fuzzy search
    kill -9 **<TAB>
    ```
    
    Select multiple processes with TAB and Shift-TAB.
  </Step>
  
  <Step title="SSH hostname completion">
    ```bash
    # Connect to hosts from ~/.ssh/config
    ssh **<TAB>
    ```
  </Step>
  
  <Step title="Environment variable completion">
    ```bash
    # Bash and Zsh
    export **<TAB>
    unset **<TAB>
    
    # Fish (use Shift-TAB)
    set <SHIFT-TAB>
    ```
  </Step>
</Steps>

## Practical Examples

### Git Workflow

<CodeGroup>
```bash Interactive Checkout
# Checkout a branch interactively
git branch | fzf | xargs git checkout
```

```bash Browse Commits
# View commit details before selecting
git log --oneline --color=always | \
  fzf --ansi --preview 'git show --color=always {1}' | \
  awk '{print $1}'
```

```bash Interactive Add
# Select files to stage
git status --short | \
  fzf -m | \
  awk '{print $2}' | \
  xargs git add
```

```bash Search Commits
# Find commits by message or content
git log --oneline | \
  fzf --preview 'git show --color=always {1}' \
      --bind 'enter:become(git show {1})'
```
</CodeGroup>

### File Operations

<CodeGroup>
```bash Find and Edit
# Find files and open in editor
fd --type f | fzf --preview 'bat --color=always {}' | xargs vim
```

```bash Search File Contents
# Interactive ripgrep
rg --files-with-matches "" | \
  fzf --preview 'bat --color=always {}' \
      --preview-window '~3'
```

```bash Delete Files
# Safely select files to delete
fd --type f | fzf -m --preview 'bat --color=always {}' | xargs rm -i
```

```bash Copy Files
# Select files to copy to destination
fd --type f | fzf -m | xargs -I {} cp {} /destination/
```
</CodeGroup>

### Process Management

<CodeGroup>
```bash Kill Process
# Interactive process killer
ps aux | fzf | awk '{print $2}' | xargs kill -9
```

```bash Docker Containers
# Stop containers interactively
docker ps | fzf | awk '{print $1}' | xargs docker stop
```

```bash System Monitor
# Monitor processes with reload
ps -ef | \
  fzf --bind 'ctrl-r:reload(ps -ef)' \
      --header 'Press CTRL-R to reload' \
      --header-lines=1
```
</CodeGroup>

### Advanced: Interactive Ripgrep

Create a powerful file content search:

```bash
# Interactive ripgrep launcher
RG_PREFIX="rg --column --line-number --no-heading --color=always --smart-case "
INITIAL_QUERY=""
: | fzf --ansi --disabled --query "$INITIAL_QUERY" \
    --bind "start:reload:$RG_PREFIX {q}" \
    --bind "change:reload:sleep 0.1; $RG_PREFIX {q} || true" \
    --delimiter : \
    --preview 'bat --color=always {1} --highlight-line {2}' \
    --preview-window 'up,60%,border-bottom,+{2}+3/3,~3' \
    --bind 'enter:become(vim {1} +{2})'
```

<Tip>
Save this as a shell function or alias for quick access. This combines ripgrep's speed with fzf's interactivity.
</Tip>

## Customization Basics

### Set Default Options

```bash
# Add to ~/.bashrc, ~/.zshrc, or ~/.config/fish/config.fish
export FZF_DEFAULT_OPTS="
  --height 40%
  --layout reverse
  --border
  --inline-info
  --preview-window 'right:60%'
  --bind 'ctrl-/:change-preview-window(down|hidden|)'"
```

### Use Better Tools for Input

```bash
# Use fd instead of find (respects .gitignore)
export FZF_DEFAULT_COMMAND='fd --type f --strip-cwd-prefix --hidden --follow --exclude .git'

# Also use for CTRL-T
export FZF_CTRL_T_COMMAND="$FZF_DEFAULT_COMMAND"

# Use fd for directory search (ALT-C)
export FZF_ALT_C_COMMAND='fd --type d --hidden --follow --exclude .git'
```

## Common Patterns

Here are some patterns to add to your shell configuration:

```bash
# fe - Open file in editor
fe() {
  local files
  files=$(fzf --query="$1" --multi --select-1 --exit-0 \
    --preview 'bat --color=always --style=numbers {}')
  [[ -n "$files" ]] && ${EDITOR:-vim} "${files[@]}"
}

# fcd - cd to selected directory
fcd() {
  local dir
  dir=$(find ${1:-.} -path '*/\.*' -prune -o -type d -print 2> /dev/null | fzf +m) &&
  cd "$dir"
}

# fkill - kill process
fkill() {
  local pid
  pid=$(ps aux | sed 1d | fzf -m | awk '{print $2}')
  [[ -n "$pid" ]] && echo "$pid" | xargs kill -${1:-9}
}

# fbr - checkout git branch
fbr() {
  local branches branch
  branches=$(git branch --all | grep -v HEAD) &&
  branch=$(echo "$branches" | fzf -d $(( 2 + $(wc -l <<< "$branches") )) +m) &&
  git checkout $(echo "$branch" | sed "s/.* //" | sed "s#remotes/[^/]*/##")
}
```
