fzf is an interactive filter program for any kind of list. It implements a "fuzzy" matching algorithm, so you can quickly type in patterns with omitted characters and still get the results you want.

## Usage

```bash
fzf [options]
```

## Quick Reference

### Common Options

```bash
# Case-insensitive search
fzf -i

# Multi-select mode
fzf -m

# With preview window
fzf --preview 'cat {}'

# Reverse layout
fzf --layout reverse

# Custom height
fzf --height 40%
```

### Environment Variables

<ParamField path="FZF_DEFAULT_COMMAND" type="string">
  Default command to use when input is tty
  
  ```bash
  export FZF_DEFAULT_COMMAND='fd --type f'
  ```
</ParamField>

<ParamField path="FZF_DEFAULT_OPTS" type="string">
  Default options applied to all fzf invocations
  
  ```bash
  export FZF_DEFAULT_OPTS="--layout=reverse --info=inline"
  ```
</ParamField>

<ParamField path="FZF_DEFAULT_OPTS_FILE" type="string">
  Location of the file to read default options from
  
  ```bash
  export FZF_DEFAULT_OPTS_FILE=~/.fzfrc
  ```
</ParamField>

<ParamField path="FZF_API_KEY" type="string">
  X-API-Key header for HTTP server (--listen)
</ParamField>
