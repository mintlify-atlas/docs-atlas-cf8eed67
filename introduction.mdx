fzf is a general-purpose command-line fuzzy finder. It's an interactive filter program for any kind of list: files, command history, processes, hostnames, bookmarks, git commits, and more. It implements a "fuzzy" matching algorithm, so you can quickly type in patterns with omitted characters and still get the results you want.

## How It Works

fzf launches an interactive finder, reads a list from STDIN, and writes the selected item to STDOUT:

```bash
# Find files and open in vim
find * -type f | fzf > selected

# Without STDIN pipe, fzf traverses the current directory
vim $(fzf)

# Use with modern tools like fd
fd --type f | fzf
```

<Tip>
You can override the default behavior by setting `$FZF_DEFAULT_COMMAND` to a command that generates your desired list, or by using `--walker`, `--walker-root`, and `--walker-skip` options.
</Tip>

## Key Features

### Fuzzy Search Syntax

fzf supports powerful search patterns out of the box:

| Token | Match Type | Description |
|-------|-----------|-------------|
| `sbtrkt` | fuzzy-match | Items that match `sbtrkt` |
| `'wild` | exact-match (quoted) | Items that include `wild` |
| `^music` | prefix-exact-match | Items that start with `music` |
| `.mp3$` | suffix-exact-match | Items that end with `.mp3` |
| `!fire` | inverse-exact-match | Items that do not include `fire` |

### Multiple Display Modes

  <Tab title="Height Mode">


  </Tab>
  <Tab title="Tmux Popup">

    ```bash

    fzf --tmux bottom,40%
    ```
  </Tab>
</Tabs>

### Preview Window

View file contents, git diffs, or any custom preview:

```bash
# Preview with bat syntax highlighting
fzf --preview 'bat --color=always {}'

# Interactive ripgrep with preview
rg --column --line-number --no-heading --color=always --smart-case "" | \
  fzf --ansi --delimiter : \
      --preview 'bat --color=always {1} --highlight-line {2}' \
      --bind 'enter:become(vim {1} +{2})'
```

## Real-World Examples

### Git Integration

```bash
# Interactive git checkout
git branch | fzf | xargs git checkout

# Browse and show commits
git log --oneline | fzf --preview 'git show --color=always {1}'

# Interactive git add
git status --short | fzf -m | awk '{print $2}' | xargs git add
```

### File Navigation

```bash
# Find and cd into directory
cd $(find . -type d | fzf)

# Open files in vim with preview
vim $(fzf --preview 'bat --color=always {}')

# Search file contents interactively
rg --files-with-matches "" | fzf --preview 'rg --color=always {}'
```

### Process Management

```bash
# Kill processes interactively
ps aux | fzf | awk '{print $2}' | xargs kill -9

# Docker container management
docker ps | fzf | awk '{print $1}' | xargs docker stop
```
